# 기술 통계 ( Descriptive Statistics )


```R
attach(trees) # 데이터 사용하기 편하게 하기
```

## 평균 ( Mean)


```R
mean(Volume)
```


30.1709677419355


## 분산 ( Variance )
    - 데이터가 표본(Sample)이면 표본분산 사용
    - 데이터가 모집단(Population)이면 모분산 사용

### 표본분산


```R
var(Volume)
```


270.202795698925


### 모분산


```R
var(Volume) * (length(Volume)-1) / length(Volume)
```


261.48657648283


## 표준편차 ( Standard Deviation )

     : 관찰치의 퍼진 정도


```R
sd(Volume)   # ( == sqrt(var(Volume)) )
```


16.4378464434647


## 표준오차 ( Standard Error )
        : 추정치의 표준편차


```R
sd(Volume) / sqrt(length(Volume))
```


2.95232437514822


### 함수 만들기


```R
se = function(x) sd(x) / sqrt(length(x))
```


```R
se(Volume)
```


2.95232437514822


## 변동계수(Coefficient of Variation)
    => 보통 평균이 증가하면 분산도 같이 증가한다  
    => ex ) 몸무게 100kg인 사람이 45kg인 사람보다 몸무게 변동성이 크다.


```R
sd(Volume) / mean(Volume)
```


0.544823307759441

## IQR(interquartile range)
    => 퍼진 정도를 측정하는 표준편차나 분산에 해당
    => Q3 - Q1


```R
IQR(Volume)
```


17.9

## Q-Q Normality plot
    => 데이터가 정규분포에 얼마나 근접한지 알고 싶으면 실행
    => qqline : 관찰값인 점들이 이 직선에서 크게 벗어나지 않으면 Volume이 정규분포를 따른다고 할 수 있다.


```R
qqnorm(Volume)
qqline(Volume , col = "blue")
```
