# Java



## - 개념

	- 고급언어 : 속도가 낮음
	- 쉬운언어 : 속도가 빠름
	-  
	- 프로그래밍 -> 컴파일러(인간이 이해하는 코드 -> 기계가 이해하는 코드) -> 기계어 실행
	- 자바(버젼 6,7,8) 이 주로 사용 -> But 최신버전도 익히도록 하자!
	
	- JDK가 설치되어야만 자바 프로그램을 개발할 수 있다.
	- JRE는 자바 실행 환경으로 자바 개발용이 아니라 자바를 사용하는 프로그램이 있는 경우 필요
	-
	- 버전관리 : 파일 변화를 시간에 따라 기록했다가 나중에 특정 시점의 버젼을 다시 꺼내올 수 있는 시스템.

## - 책

```
  - 프로그래밍 언어를 배워야 하는 이유
    - 사람이 하기 어려운 문제나 , 반복적으로 지루한 작업을 자동으로 처리하기 위해서
  - 자바의 특징
    - 객체지향언어
    - JVM(Java Virtual Machine)
    - Write Once , Run Anywhere => 한번 쓰면 어디서든 실행 가능!
  - 자바 프로그램 종류
    - JRE(Java Runtime Environment) : 자바로 만들어진 프로그램을 실행하기 위한 프로그램
    - JDK(Java Development Kit) : 자바 개발을 위한 프로그램
  - 자바 버젼
    - SE(Standard Edition) : 자바 표준 버젼 , PC or Android용 프로그램 개발을 위한 버젼
    - EE(Enterprise Edition) : Web Application을 개발하기 위한 버젼
  - 주석 : 프로그램 내에서 실행되지 않는 영역을 표시할 때 사용.
  - 실행문 : 프로그램에서 처리하는 하나의 단위 -> 자바에서는 ; 을 사용.
  - 코드를 작성할 때 누가 봐도 알아보기 쉽게 코딩하는 것이 좋다.

  - 변수 이름 짓는 규칙
    - 1 . 알파벳 , 숫자 , 특수문자 , 한글 사용 가능.
    - 2 . 숫자로 시작 x.
    - 3 . 공백 사용 불가.
    - 4 . 대/소문자를 구분.
    - 5 . 예약어 사용 불가.

  - 자바 규칙
    - 1 . 클래스명은 첫 글자를 대문자로.
    - 2 . 변수명이 여러 단어일 경우 Camel 표기법 or snake 표기법을 사용.
    - 3 . 상수의 변수명은 모두 대문자로.
  - 자동 형변환 : 프로그램 실행 중에 자동으로 자료형이 변환되는 것.
  - 강제 형변환 : 값을 대입하기 전에 괄호로 변환하고자 하는 대상의 자료형 이름을 넣어줘야 한다.
```
