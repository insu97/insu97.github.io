```java
public class HelloWorld {														
    public static void main(String[] args) {									 
        System.out.println("Hello World!");
    }
}

// public class HelloWorld : 'HelloWorld' 라는 클래스 정의
// public : 접근 제어자
// void : 메소드(함수)의 리턴 값의 자료형(void는 리턴값 없음)
// main : 메소드(함수) 이름 , args : 파라미터 (args라는 문자열 배열)
// Systemp이라는 클래스의 , out이라는 변수의 , println이라는 메서드 호출
```
