1 . 개념

- 오차행렬

- | 실제/예측 |      |      |      |
     | :-------- | ---- | ---- | ---- |
     |           |      | 음성 | 양성 |
     |           | 양성 | TN   | FP   |
     |           | 음성 | FN   | TP   |

     - 정밀도 = TP / (TP + FP)
     - 재현율 = TP / (TP + FN)
     - F1 = 2 / (1/정밀도 + 1/재현율)
     - 상황에 따라 정밀도가 중요할 수 도 재현율이 중요할 수도..!
     - '정밀도/재현율 트레이드 오프' : 정밀도와 재현율은 반비례 관계
