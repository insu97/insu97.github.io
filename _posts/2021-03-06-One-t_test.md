# 검정 방법

## One-Sample T-test

​	: 한 모집단의 평균이 어떤 특정한 값과 같은지 검증하는 통계 방법


```R
x = c(15.5 , 11.21 , 12.67 , 8.87 , 12.15 , 9.88 , 2.06 , 14.5 , 0 , 4.97)  # 2008년 데이터 , 2006년 평균 = 8.1
mean(x)
```

9.181

```R
sd(x) # 표준편차
```

5.23496460775471

```R
mean(x) - 8.1   # if 모평균 = 8.1
```

1.081

```R
sqrt(length(x))*(mean(x)-8.1)/sd(x) # 검정 통계량 ( = t값 )
```


0.652998139773134

```R
# ==> 자유도가 9(=n-1)인 t분포를 따른다
2*(1-pt(0.653 , df = 9)) # p-value( = 유의 확률 )
```


0.530081783299648

```R
# 정규성검정
shapiro.test(x)     # p-value > 0.05 => 데이터가 정규분포를 따른다는 귀무가설을 기각하지 못한다.
```

    	Shapiro-Wilk normality test

    data:  x
    W = 0.92341, p-value = 0.3863


```R
t.test(x , mu = 8.1)                                                      # p-value > 0.05 => 평균이 8.1이라는 귀무가설을 기각하지 못한다.
# t.test(x , mu = 8.1 , conf.level = 0.99)                                # 유의확률을 0.99로 설정
# t.test(x , mu = 8.1 , alter = "greater")                                # mu > 8.1
# t.test(x , mu = 8.1 , alter = "less")                                   # mu < 8.1
```

    	One Sample t-test

    data:  x
    t = 0.653, df = 9, p-value = 0.5301
    alternative hypothesis: true mean is not equal to 8.1
    95 percent confidence interval:
      5.436132 12.925868
    sample estimates:
    mean of x
        9.181


```R
out = t.test(x , mu=8.1)
```
