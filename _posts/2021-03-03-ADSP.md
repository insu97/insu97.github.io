## 데이터

1 . 데이터 종류

  - 정성 데이터  : 많은 비용이 소모 ( ex . 언어 , 문자 )  => 처리하기 어려운 데이터.
  - 정량 데이터  : 비용 소모가 적음 ( ex . 수치 , 도형 , 기호 ) => 비교적 처리하기 쉬운 데이터

2 . 지식경영의 핵심

  - 암묵지 : 개인의 지식(내면화) => 조직의 지식으로(공통화)
  - 형식지 : 집단의 지식(표출화) => 개인의 지식으로(연결화)

3 . 개인정보 비식별 기술

  - 데이터 마스킹 : 홍길동 -> 홍**
  - 가명처리  : 홍길동 -> 임꺽정
  - 총계처리  : 홍길동 180cm , 임꺽정 175cm ... => 학생 평균 키 : 165cm
  - 데이터값 삭제 : 홍길동 , 남자 , 35세  => 남자 , 35세
  - 데이터 범주화 : 홍길동 35세 => 홍씨 , 30~40세

4 . 데이터의 유형

  - 정형데이터 : 관계형 데이터 베이스에 저장
  - 반정형데이터 : 파일로 저장
  - 비정형데이터 : NOSQL에 저장

5 . DIKW

  - Data : 개별 데이터 자체로는 의미가 중요하지 않음.
  - Information : 데이터의 가공 , 처리와 데이터간 연관관계 속에서 의미가 도출된 것.
  - Knowledge : 결론과 개인적인 경험을 결합시켜 고유의 지식으로 만듬.
  - Wisdom : 지식의 축적과 아이디어가 결합된 창의적인 산물.

6 . 빅데이터 활요에 필요한 기본적인 3요소

  - 데이터
  - 기술
  - 인력

7 . 빅데이터가 만들어 내는 본직적인 변화

  - 사전처리 -> 사후처리
  - 표본조사 -> 전수조사
  - 질 -> 양
  - 인과관계 -> 상관관계

8 . 빅데이터 활용 분석

  - 연관규칙학습
  - 유형분석
  - 유전자 알고리즘
  - 기계학습
  - 회귀분석
  - 감정분석
  - 소셜네트워크 분석(= 사회관계망분석 )

9 . 데이터 사이언티스트의 요구역량
  - 빅데이터에 대한 이론적 지식
  - 분석 기술에 대한 숙련
  - 통찰력 있는 분석
  - 설득력 있는 전달
  - 다분야간 협력

0 . 용어
  - OLTP(On-Line Transaction Processing) : 호스트 컴퓨터가 DB를 엑세스하고 , 바로 처리 결과를 돌려보내는 형태.
  - OLAP(On-Line Analytical Processing) : 다차원적인 데이터에 접근하여 의사 결정에 활용할 수 있는 정보를 얻을 수 있게 해주는 기술.
  - CRM(Customer Relationship Management) : '고객관계관리'
  - SCM(Supply Chain Management) : '공급망 관리'
  - BA(Business Analytics) : 의사결정을 위한 통계적이고 수학적인 분석에 초점을 둔 기법.
  - BI(Business Intelligence) : 데이터 기반 의사결정을 지원하기 위한 리포트 중심의 기법.
  - ERP(Enterprise Resource Planning) : 기업 전체를 통합적으로 관리하고 경영의 효율화를 기하기 위한 시스템.
  - 메타 데이터 : 다른 데이터를 설명해 주는 데이터.
