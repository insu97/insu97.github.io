```java
public class HelloWorld {														
    public static void main(String[] args) {									 
        System.out.println("Hello World!");
    }
}

// public class HelloWorld : 'HelloWorld' 라는 클래스 정의
// public static void main(String[] args) : 클래스가 어떤 동작을 할 지 정의 => 메소드 (=클래스 안에 정의된 함수 )
// public : 누구나 사용가능 , 접근 제어자(ex . private , protected)
// void : 메소드(함수)의 리턴 값의 자료형(void는 리턴값 없음) , (int -> 숫자 리턴)
// main : 메소드(함수) 이름 , 가장 먼저 실행
// static : 바로 실행 가능하게 만들어줌 , main앞에는 항상 static을 붙이게 됩니다
// String[] args : 전달되는 값 , 파라미터 , string[] : 문자열 , args : 변수 이름
// System : 자바에서 미리 정해둔 클래스 , 입력 및 출력에 관련된 기능들을 모아둔 클래스
// out : 출력과 관련된 기능
// println : print + line , 줄 출력 (= 메소드)
// print : 줄바꿈 없음
// printf : 자유롭게 작성
```

==> 출력 :  Hello World!

```java
public class variables {
    public static void main(String[] args) {
        int age;
        age = 24;
        System.out.printf("내 나이는 %s살 입니다.%n" , age);
    }
}

// 변수선언 : type variableName;
// 변수 이름 규칙
// 1 . 대소문자를 구분
// 2 . 숫자로 시작할 수 없다.
// 3 . 밑줄(_)과 달러 표시($)를 사용할 수는 있다
//	=> but 사용하지 않는것이 좋다. => why? $는 주로 자동 생성되는 변수명으로 쓰이고 , _는 보통 상수 이름에 쓰인다.
// 4 . class , public 과 같은 자바의 예약어는 변수명으로 쓸 수 없다.
// 5 . 자바의 변수명은 '카멜 케이스(camelCase)'라는 기법으로 작성하는게 좋다.
//	=> 첫 번째 글자는 소문자 , 그 후에 새로운 단어의 첫 번째 글자는 대문자 사용(ex . myName)
```

==> 출력 : 내 나이는 24살 입니다.
