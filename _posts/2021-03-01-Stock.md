- CMO(Contract Manufacturing Organization , )
  - : 의약품 위탁생산업체
- IPO(Initial Public Offering)
  -  : 비상장기업이 증권시장에서 공식적으로 거래되기 위해 '기업공개'라는 과정을 뜻함.
- OEM (Original Equipment Manufacture) 
  - : 완제품을 제조*공급하는 , 일종의 하청부 생산
- ODM(Original Development Manufacturing , '제조사개발생산')
  - : 주문자의 요구에 따라 ODM 업체가 주도적으로 제품을 생산
- PBR = 시가총액 / 내 재산
  - => PBR이 낮으면 주가가 cold , 땅이나 현금을 갖고 있는 경우가 많다
  - => PBR이 높으면 주가가 hot , But 위험이 높음
- 기업공개 
  - : 외부 투자자가 주식을 살 수 있도록 기업이 자사의 주식과 경영내역을 시장에 공개하는 것.
- 기회비용
  - : 내가 갖고 있는 주식의 기대수익보다 다른 주식의 기대수익이 높다면 매도 후 타 기업을 매수
- 매출 채권 회전율
  -  : 외상을 했을 때 돈이 잘 들어오는지 => 높을수록 좋음.
- 버핏지수
  - : 국내총생산(GDP) 대비 시가총액 비율을 뜻 함.
  - &nbsp;(ex . 버핏지수가 100% : GDP = 시가총액 , 70% : 저평가 , 120% : 고평가)
- 상저하고 
  - : 상반기 하락 , 하반기 상승
- 시크리컬(Cyclical , 경기민감주) 
  - : 경기 사이클에 민감하게 반응하는 업종을 뜻 함&nbsp;(ex . 해운사 , 조선사 )
  - &nbsp; 1 . 원달러환율이 하락 -> 시크리컬에 투자.
  - &nbsp; 2 . EU의 경기가 회복될 때 -> 시크리컬에 투자.
- 시가총액
  - : 주가 * 기업이 발행한 총 주식 수
  - 시가총액은 중요한 개념 => 주가만 신경쓰지 말고 시가총액도 보자
- 유상증자
  - : 기업이 자금을 조달하기 위해 주식시장 내에서 투자자들을 대상으로 주식을 발행하는 행위
  - : 유상증자 발생 => 악재 => '당기순손실' 인 기업들이 유상증자를 주로 발행
- 액면분할
  - : 접근성이 높아져서 '호재' 라고 생각해도 됨.
- 이자보상배율 
  - : 부채를 갚을 수 있는 능력
  - : 높으면 높을 수록 좋음
  - ( Tip . 이자보상배율이 1이하 기업이면 부채를 갚을 능력이 없다고 판단)
- 역배열
  - : 5,20,60,120 이평선이 역으로 배열되어있는 상태 ( => 하락추세 )
- 재고 자산 회전율
  - : 재고가 빨리 나가는지 => 높을수록 좋음
- 정배열
  - : 5,20,60,120 이평선이 차례로 배열되어있는 상태 ( => 상승추세 )

Tip . 가치함정에 빠지지 말자! => 시간이 지나도 안오르는 주식은 관심을 못 받는 주식일 수 있다.

Tip . 최대 주주 지분이 낮을 때 기업가치가 낮으면 위험. ( ex . 최대주주 지분 10% -> 낮다고 판단)

=====> 최대 주주가 사면 주가가 싸고 , 팔면 주가가 비싸다고 판단.

Tip . '주식관련사채'를 발행하는 기업은 않좋다.

Tip. PER = 시가총액 / 영업이익(당기순이익)

​	=> PER         => '과거' 데이터

​	=> 추정 PER => '미래'를 예측하여 추정

Tip . 부채비율 : 100% 이하 => 현금으로 부채를 갚을 수 있다.
Tip . 당좌비율 : 1년안에 들어올 현금과 부채를 비교 (현금/부채)

​	=> 100% 이상이면 Good

​	=> 100% 이하이면 유상증자 => Bad

Tip . 유보율 : 잉여금 / 자본금

​	=> 유보율이 높으면 -> 무상증자 -> Good

Tip . 시가배당률 : 1월에 '고배당주' Buy -> 12월에 배당금 Get or 고점이면 Sell.

Tip . 물적분할 : 기업이 분할되어 나가서 새로운 기업 상장 -> 내 생각에는 악재

Tip . 장기투자를 하려면 기준은 ROE가 10년 , 5년 , 3년 이상 꾸준히 10%이상인 기업
