2. 용어 정리  

   - OEM (Original Equipment Manufacture) : 완제품을 제조*공급하는 , 일종의 하청부 생산<br>
   - 이자보상배율 : 부채를 갚을 수 있는 능력<br>
   &nbsp;&nbsp;=> 높으면 높을 수록 좋음<br>
   &nbsp;&nbsp;=> ( Tip . 이자보상배율이 1이하 기업이면 부채를 갚을 능력이 없다고 판단)<br>
   - 상저하고 : 상반기 하락 , 하반기 상승<br>
   - 시가총액 : 주가 * 기업이 발행한 총 주식 수<br>
   &nbsp;&nbsp;=> 시가총액은 너무나 중요한 개념 => 주가만 신경쓰지 말고 시가총액도 보자<br>
   - 정배열 : 5,20,60,120 이평선이 차례로 배열되어있는 상태 ( => 상승추세 )<br>
   - 역배열 : 5,20,60,120 이평선이 역으로 배열되어있는 상태 ( => 하락추세 )<br>
   - 기업공개 : 외부 투자자가 주식을 살 수 있도록 기업이 자사의 주식과 경영내역을 시장에 공개하는 것.<br>
   - IPO(Initial Public Offering)<br>
   &nbsp;&nbsp;: 비상장기업이 증권시장에서 공식적으로 거래되기 위해 '기업공개'라는 과정을 뜻함.<br>
   - 유상증자 : 기업이 자금을 조달하기 위해 주식시장 내에서 투자자들을 대상으로 주식을 발행하는 행위<br>
   &nbsp;&nbsp;=>유상증자 -> '악재' - > '당기순손실'인 기업들이 유상증자를 주로 발행<br>
   - CMO : 의약품 위탁생산업체<br>
   - OEM(Original Equipment Manufacturing , '주문자상표부착생산')<br>
   &nbsp;&nbsp;: 고객사가 원하는 대로 제품 납품.<br>
   - ODM(Original Development Manufacturing , '제조사개발생산')<br>
   &nbsp;&nbsp;: 주문자의 요구에 따라 ODM 업체가 주도적으로 제품을 생산<br>
   - 시크리컬(Cyclical , 경기민감주) : 경기 사이클에 민감하게 반응하는 업종을 뜻 함.<br>
   &nbsp;&nbsp;(ex . 해운사 , 조선사 )
   &nbsp;&nbsp; 1 . 원달러환율이 하락 -> 시크리컬에 투자.<br>
   &nbsp;&nbsp; 2 . EU의 경기가 회복될 때 -> 시크리컬에 투자.<br>
   - 버핏지수 : 국내총생산(GDP) 대비 시가총액 비율을 뜻 함.<br>
   &nbsp;&nbsp;(ex . 버핏지수가 100% : GDP = 시가총액 , 70% : 저평가 , 120% : 고평가)<br>
   - 매출 채권 회전율 : 외상을 했을 때 돈이 잘 들어오는지 => 높을수록 좋음.<br>
   - 재고 자산 회전율 : 재고가 빨리 나가는지 => 높을수록 좋음.<br>
   - PBR = 시가총액 / 내 재산<br>
     - => PBR이 낮으면 주가가 cold , 땅이나 현금을 갖고 있는 경우가 많다.<br>
     - => PBR이 높으면 주가가 hot , But 위험이 높음<br>
   - (E) : 예상<br><br>


3. 심리  

   - 만약 본인이 좋은 기업이라 판단 했을 때 주가가 갑자기 폭락 한다면 바로 팔지말고 신중하게 결정해야 한다.<br>
   &nbsp;&nbsp; => 개인 투자자가 손해를 보는 경우.<br>
   - 장기 투자자들은 하루하루에 연연하지말아야 한다.<br>
   - 기회비용 : 내가 갖고 있는 주식의 기대수익보다 다른 주식의 기대수익이 높다면 매도 후 타 기업을 매수<br>
   - 한 업종이 불황이라 관련 회사가 사라지고 있다면 주도주에 투자해라!<br>
   =======> 업종이 망하지 않는한 주도주에 경쟁사가 사라짐.<br>
   =======> 한동안 주가가 떨어질 수 있지만 향후 길게 본다면 투자를 고려!<br>
   - 좋은 기업을 매수 했다면 수면제먹고 자라!<br>
   - 브랜드를 쌓는 기업들은 시간이 지날수록 좋아진다.(ex .빅데이터)<br><br>

4. 기법  

   - PER = 시가총액 / 영업이익(당기순이익)<br>
     - PER => '과거' 데이터<br>
     - 추정 PER => '미래'를 예측하여 추정<br>
   - 액면분할 : 접근성이 높아져서 '호재' 라고 생각해도 됨.<br>
   - 호재 관련 뉴스가 나온다면<br>
     1.  주가가 많이 오른 상태<br>
     &nbsp;&nbsp;=> 상승이 잘 일어나지 않는다.<br>
     &nbsp;&nbsp;(why ? 주가는 이미 선반영이 되어있을 확률이 높기 때문에)<br>
     &nbsp;&nbsp;=> 따라서 호재가 나온날 보고 사면 손해를 볼 확률이 높다.<br>
     2.  박스권<br>
         => 상승이 나올 확률이 있다.<br>
         => 하지만 호재가 나오고 주가를 보면 높은 가격에 사야될 확률이 높기 때문에 안 사는걸 추천.<br>
   - 최대 주주 지분이 낮을 때 기업가치가 낮으면 위험. ( ex . 최대주주 지분 10% -> 낮다고 판단)<br>
   - 최대 주주가 사면 주가가 싸고 , 팔면 주가가 비싸다고 판단.<br>
   - 당기순이익이 늘어날 기업을 찾자. => '뉴스'로 확인<br>
   - '실적개선'이 되는 기업을 찾자. => 저평가 => 성장가능성이 높다.(장기투자) <br>
   - '주식관련사채'를 발행하는 기업은 않좋다.<br>
   - 부채비율 : 100% 이하 => 현금으로 부채를 갚을 수 있다.<br>
   - 당좌비율 : 1년안에 들어올 현금과 부채를 비교 (현금/부채)<br>
     - 100% 이상이면 Good<br>
     - 100% 이하이면 유상증자 => Bad<br>
   - 유보율 : 잉여금 / 자본금<br>
     - 유보율이 높으면 -> 무상증자 -> Good<br>
   - 시가배당률 : 1월에 '고배당주' Buy -> 12월에 배당금 Get or 고점이면 Sell.<br>
   -  물적분할 : 기업이 분할되어 나가서 새로운 기업 상장 -> 내 생각에는 악재<br>
